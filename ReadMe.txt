# Praveen's Website

In this repository is my website, "Praveen's Website".

## Updates from Latest release
- CSS Style sheet implemented, consistent styling of elements
- New font, color scheme, aesthetics
- Re-arrangement of various elements into new units for enhanced flexbox display (wrapper, container)
- Responsiveness of design on mobile and print views

## Contents

In this repository, there are HTML files for the first six pages. The content of pages includes descriptions about myself, my hobbies, interests and inspirations, as well as contact information and release history.

Home: Introductory page, mainly with writing about my background.
Activities: Gallery of artwork.
Interests: Additional content about my interests.
Media: Additional content I appreciate.
Release History: Table of release information records.

The image and video content of the site are stored in the repository as files, in the media folder. They are referenced by the site's HTML code.

## View

To view the site, open index.html in a web browser.
The site can be navigated by clicking the navigational links at the top of each page.

To view as mobile: Select the browser's "Inspect tool" -> Mobile emulator
To view as print: Select the browser's "Print" option (or CTRL+P) and view preview
